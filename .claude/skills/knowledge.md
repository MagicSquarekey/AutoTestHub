# Knowledge Index Skill

## 触发条件
当用户问"这个项目"、"项目结构"、"怎么实现"时触发。

## 功能
搜索项目知识库，提供相关信息。

## 执行流程
1. 理解用户问题
2. 搜索相关文档
3. 提取关键信息
4. 提供答案

## 知识来源
1. **CLAUDE.md** — 项目规范
2. **shi_yong_shou_ce.md** — 使用手册
3. **zhi_shi_ku.md** — 知识库索引
4. **代码注释** — 代码说明
5. **lessons.md** — 经验记录

## 输出
搜索结果应包含：
- 相关文档链接
- 关键信息摘要
- 使用示例
- 相关建议

## 使用场景
- 了解项目结构
- 查找使用方法
- 理解设计决策
- 解决常见问题
