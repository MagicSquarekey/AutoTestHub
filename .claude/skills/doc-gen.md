# Document Generation Skill

## 触发条件
当用户说"写文档"、"更新文档"、"生成文档"时触发。

## 功能
自动生成或更新项目文档。

## 执行流程
1. 分析需要文档的内容
2. 检查现有文档结构
3. 生成或更新文档
4. 确保文档完整性

## 文档规范
- 人看的文档用拼音命名
- AI 看的文档用英文命名
- 文档内容清晰易懂
- 包含必要的示例

## 输出
生成的文档应包含：
- 清晰的标题
- 详细的说明
- 使用示例
- 相关链接

## 常见文档
- README.md — 项目简介
- shi_yong_shou_ce.md — 使用手册
- API 文档
- 代码注释
