# Error Recovery Skill

## 触发条件
当测试失败、代码报错或出现异常时自动触发。

## 功能
自动分析错误并尝试修复。

## 执行流程
1. 捕获错误信息
2. 分析错误原因
3. 定位问题代码
4. 提供修复方案
5. 尝试自动修复

## 错误类型
1. **语法错误** — 代码语法问题
2. **运行时错误** — 代码执行问题
3. **逻辑错误** — 代码逻辑问题
4. **依赖错误** — 依赖缺失或版本问题

## 输出
分析结果应包含：
- 错误描述
- 错误原因
- 修复方案
- 预防建议

## 自动修复
对于简单错误，可以尝试自动修复：
- 语法错误：自动修正
- 导入错误：自动添加导入
- 格式错误：自动格式化
