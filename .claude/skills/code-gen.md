# Code Generation Skill

## 触发条件
当用户说"写代码"、"实现"、"编写"、"创建"时触发。

## 功能
根据项目规范自动生成代码。

## 执行流程
1. 理解用户需求
2. 检查 CLAUDE.md 中的代码规范
3. 生成符合规范的代码
4. 添加必要的注释和文档

## 代码规范
- 文件头部：`# -*- coding: utf-8 -*-`
- 函数必须有 docstring
- 参数和返回值要有类型提示
- 职责单一，不超过 50 行
- 缩进：4 个空格
- 行宽：最大 120 字符

## 输出
生成的代码应包含：
- 完整的函数实现
- 详细的 docstring
- 类型提示
- 必要的注释
