# /img — 图片识别

通过外部工具识别图片，然后把结果告诉 AI。

## 使用方式

```
/img                # 提示用户如何操作
/img [图片描述]     # 基于描述分析图片
```

## 工作流程

### 步骤 1：保存图片
将截图保存到本地，例如：`screenshot.png`

### 步骤 2：上传到多模态工具
访问以下任一工具：
- https://claude.ai （推荐，支持中文）
- https://www.doubao.com （豆包，国产推荐）
- https://chat.openai.com （GPT-4V）
- https://gemini.google.com （Gemini）

### 步骤 3：获取分析结果
在工具中输入：
```
请详细描述这张图片的内容，包括：
1. 图片类型（截图、照片、图表等）
2. 主要内容和元素
3. 文字内容（如果有）
4. 布局和结构
5. 其他重要细节
```

### 步骤 4：复制结果给我
将分析结果复制，粘贴给我即可。

## 示例

```
你：帮我看看这张图片
AI：请按照以下步骤操作：
    1. 将图片保存到本地
    2. 访问 https://claude.ai
    3. 上传图片并输入："请详细描述这张图片的内容"
    4. 复制分析结果，粘贴给我

你：[粘贴分析结果]
AI：我来基于这个描述帮你分析...
```

## 快捷命令

如果用户发送了图片描述，直接基于描述分析：
```
你：这是一张登录页面的截图，包含用户名和密码输入框
AI：我来帮你分析这个登录页面...
```

## 注意事项

- mimo-v2.5-pro 不支持图片识别
- 需要借助外部工具完成图片分析
- 把分析结果以文字形式发给我即可
- 支持多张图片：分别分析后一起发给我
